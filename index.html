Projeto com ESP32

Automação de uma garra pneumática utilizando sensores e controle eletrônico.

Sobre o Projeto

Este site apresenta o funcionamento de um sistema de automação desenvolvido no SENAI, utilizando uma garra pneumática controlada por ESP32 para manipulação de peças em uma linha de produção.

Tecnologias Utilizadas

Foram utilizadas tecnologias como ESP32, sensores industriais, atuadores pneumáticos e programação embarcada para controle e automação do sistema.

Objetivo do Projeto

Desenvolver um sistema de separação automática capaz de identificar e classificar peças reflexivas e não reflexivas em uma esteira, simulando um processo industrial de seleção.

Equipamentos

ESP32, módulo relé, DOIT DevKit V1, sensores, atuadores pneumáticos, regulador de pressão, além de ferramentas de desenvolvimento como VS Code, PlatformIO e GitHub.

Cuidados de Segurança

Evitar o uso de adornos, manter atenção com o funcionamento do sistema, não operar a garra sem necessidade e ter cuidado com a pressão dos componentes pneumáticos.

Lógica do Sistema

Quando uma peça é detectada na esteira, a garra realiza a captura e, por meio dos sensores, identifica seu tipo. Em seguida, a peça é direcionada automaticamente para a esteira correta conforme sua classificação.
